Vagrant 设置
============

Vagrant 使用
------------
>安装完 git 后，右击鼠标选择“Git Bash Here” 打开 git。 通过 git 使用 vagrant， vagrant 是以 CLI 的形式操作。

Vagrant 常用指令
----------------
**vagrant init**
```
对 vagrant 初始化
```
**vagrant status**
```
查看 vagrant 状态
```
**vagrant global-status**
```
查看 vagrant 全局状态
```
**vagrant box list**
```
查看当前所有 vagrant box 列表
```
**vagrant box add box_name path/add.box**
```
新增 vagrant box 名称 路径
```
**vagrant up**
```
打开 vagrant 虚拟机
```
**vagrant ssh**
```
登录 vagrant 虚拟机
```
**vagrant halt**
```
先关闭虚拟机
```
